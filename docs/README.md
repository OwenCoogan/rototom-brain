# Documentation

This directory contains all documentation for the Rototom Brain project.

## Structure

- **FR/** - French documentation
- **EN/** - English documentation

## French Documentation (FR)

- **ACTIVER_IAC_FR.md** - Guide to activate IAC Driver on macOS (French)
- **GUIDE_PONT_PYTHON_FR.md** - Complete guide to use Python MIDI bridge (French)
- **IAC_DRIVER_FR.md** - IAC Driver setup and troubleshooting (French)
- **VERIFICATION_MIDI_FR.md** - How to verify native USB MIDI (French)
- **VERIFIER_GARAGEBAND_FR.md** - How to verify MIDI in GarageBand (French)

## English Documentation (EN)

- **MIDI_SETUP.md** - Complete MIDI setup guide for GarageBand and DAWs
- **MIDI_TROUBLESHOOTING.md** - Troubleshooting guide for MIDI issues
